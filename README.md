# Swiggy-Sales-Analysis-using-Python

## Project Overview

This project focuses on analyzing Swiggy sales data using Python to extract meaningful business insights and visualize sales performance across different dimensions.

The analysis includes data cleaning, exploratory data analysis (EDA), KPI calculation, trend analysis, and visualization to help understand customer behavior and sales performance.

## Business KPIs

* Total Sales
* Average Rating
* Average Order Value
* Total Orders
* Ratings Count

## Analysis Performed

* Monthly Sales Trend
* Daily Sales Trend
* Weekly Trend Analysis
* Quarterly Performance Summary
* Top 5 Cities by Sales
* Sales by Food Type (Veg vs Non-Veg)
* State-wise Sales Analysis

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Key Insights

* Identified top-performing cities based on revenue.
* Compared sales contribution of Veg and Non-Veg food categories.
* Analyzed weekly and monthly revenue fluctuations.
* Evaluated customer ratings and order patterns.
* Built KPI metrics for business performance monitoring.

## Conclusion

This project demonstrates practical data analysis skills including data cleaning, transformation, aggregation, visualization, and deriving actionable business insights using Python. It showcases the ability to convert raw data into meaningful reports for decision-making.
